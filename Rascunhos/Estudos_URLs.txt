O Arquivo 'urls.py' é a porta de entradas da nossa aplicação web.

É com ela que os usuários conseguem encontrar suas paginas dentro 
do seu site.


formato do arquivo:
    """
    from django.contrib import admin
    from django.urls import path

    urlpatterns = [
        path('admin/', admin.site.urls),
    ]
    """



Por exemplo: (Queremos acessar dentro do nosso site a página sobre)

    CAMINHO DO NAVEGADOR:
        http://127.0.0.1:8000/sobre

    NO ARQUIVO:
        urlpatterns = [
            path('admin/', admin.site.urls),
            path('sobre/', view.sobre, name='sobre')
        ]


MOVER AS URLS PARA DENTRO DO APLICATIVO:

    from django.contrib import admin
    from django.urls import path, include


    urlpatterns = [
        path('admin/', admin.site.urls),
        path('', include('recipes.urls')), # dominino/...
        path('recipes/', include('recipes.urls')), # dominino/recipes/...
    ]
    

PARA FAZER UMA URL DINAMICA:

    urlpatterns = [
        path('', views.home), # Home
        path('recipes/<int:id>/', views.recipe), # Recipe
    ]

    Em django o parâmetro id(<int:id>) será automaticamente enviado commo
    parâmetro para o views.recipe. Assim poderemos acessar paginas unicas
    de um produto ou receita.

    https://docs.djangoproject.com/en/4.2/ref/urls/

    def recipe(request, id):
        return render(request, 'recipes/pages/home.html', context={'nome': 'Vitor Kruel'})
